# C언어 while 반복문

## 구조
```c
while(조건){
    실행문
}
```
[AD]
예시
--
```c
int a = 1;
while(a <= 10)  //a가 10 이하가 참일 때까지 반복
{
    printf("현재 a의 값: %d\n", a);
    a++;        //a에 +1
}
```

결과
--
```c
현재 a의 값: 1
현재 a의 값: 2
현재 a의 값: 3
현재 a의 값: 4
현재 a의 값: 5
현재 a의 값: 6
현재 a의 값: 7
현재 a의 값: 8
현재 a의 값: 9
현재 a의 값: 10
```

while의 무한반복
--
```c
while(1)
{
    printf("무한반복");
}
```

결과
--
```c
무한반복
무한반복
무한반복
...
```


Break
--
특정 조건에서 반복문을 강제로 종료할 때 사용.

break 사용 예시
--
```c
int a = 1;
while(a <= 10) {
    if(a == 5) {
        break; // a가 5면 반복 종료
    }
    printf("%d\n", a);
    a++;
}
```
결과
--
```c
1
2
3
4
```

Continue
--
반복문에서 현재 반복만 건너뛰고 다음 반복으로 넘어갈 때 사용.

continue 사용 예시
--
```c
int a = 0;
while(a < 10) {
    a++;
    if(a % 2 == 0) {
        continue; // 짝수면 출력 건너뛰기
    }
    printf("%d\n", a);
}
```

결과
--
```c
1
3
5
7
9
```
