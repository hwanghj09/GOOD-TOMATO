# C언어 입력

C언어에서 입력은 `scanf`을 사용한다

정수
--
```c
int a;
printf("숫자를 입력하세요 : ");
scanf("%d", a);
printf(a);
```

결과
--
```c
숫자를 입력하세요 : 4   //4 입력
4
```

[AD]
문자열
--
```c
char name[15];
printf("이름을 입력하세요: ");
scanf("%s", name);
printf("당신의 이름은 %s\n", name);
```

결과
--
```c
이름을 입력하세요 : 멋쟁이 토마토    //사용자 입력
당신의 이름은 멋쟁이
```

C언어의 `scanf`는 공백을 **문자열 구분자**로 인식한다.

문자열 띄어쓰기 해결방법
--

fget 사용
--
```c
char name[50];
printf("이름을 입력하세요: ");
fgets(name, sizeof(name), stdin);
printf("입력한 이름: %s", name);
return 0;
```

결과
--
```c
이름을 입력하세요: 멋쟁이 토마토    //사용자 입력
입력한 이름: 멋쟁이 토마토
```

`sizeof(name)` -> 배열 전체의 크기

`stdin` -> 자동으로 \0(널 문자) 추가

[^\n] 추가
--
```c
char name[15];
printf("이름을 입력하세요: ");
scanf(" %[^\n]", name);
printf("당신의 이름은 %s\n", name);
```

결과
--
```c
이름을 입력하세요: 멋쟁이 토마토    //사용자 입력
입력한 이름: 멋쟁이 토마토
```

`" "` -> 앞의 공백 문자 무시

`"[^\n]"` -> 엔터 전까지 모두 읽음
