# C언어 기본프로젝트

지금까지 배운 것들을 토대로 상품관리프로그램을 제작할 것이다.

제일 먼저 메인 화면에는

```
--------------상품 관리--------------
1. 구매자 모드
2. 관리자 모드
3. 나가기
-----------------------------------
입력하세요 : 
```
---
메인 화면에서 1번(구매자 모드)을 입력하면
# 구매자 모드
```
-------------구매자 모드-------------
1. 상품 구매
2. 잔액 확인
3. 돈 추가
4. 나가기
-----------------------------------
입력하세요 : 
```

구매자 모드에서 1번(상품 구매)을 입력하면

```
--------------상품 구매--------------
1. 사과 5000원 5개
2. 망고 4000원 4개
3. 토마토 2000원 3개
------------------------------------
현재 잔액 : 50000
입력하세요 : 
```
각 상품과 가격, 재고는 배열에 저장되어있음

현재 잔액은 변수로 저장되어있음

입력 시 잔액에서 돈이 가격만큼 줄어들고,

만약 잔액이 부족하거나 상품의 재고가 없으면 결제가 되지 않음

---

구매자 모드에서 2번(잔액 확인)을 입력하면

```
-------------잔액 확인--------------
현재 잔액 : 50000
-----------------------------------
```
현재 잔액을 보여주고 구매자 모드로 다시 이동함

---

구매자 모드에서 3번(돈 추가)을 입력하면
```
-------------돈 추가----------------
얼만큼 돈을 추가할까요?
-----------------------------------
입력하세요 : 50000

-------------잔액 확인--------------
현재 잔액 : 100000
-----------------------------------
```
입력한 만큼 돈을 추가하고, 현재 잔액을 보여줌

---

구매자 모드에서 4번(나가기)을 입력하면
관리자 모드로 이동함

---

메인 화면에서 2번(관리자 모드)을 입력하면

# 관리자 모드
```
-------------관리자 모드-------------
1. 상품 목록 확인
2. 상품 추가
3. 상품 수정
4. 상품 삭제
5. 재고 추가
6. 나가기
-----------------------------------
입력하세요 : 
```
---

관리자 모드에서 1번(상품 목록 확인)을 입력하면

```
--------------상품 목록--------------
1. 사과 5000원 5개
2. 망고 4000원 4개
3. 토마토 2000원 3개
------------------------------------
입력하세요 : 
```
상품 목록을 보여주고 바로 관리자 모드로 다시 이동함

---

관리자 모드에서 2번(상품 추가)을 입력하면
```
--------------상품 추가--------------
1. 사과 5000원
2. 망고 4000원
3. 토마토 2000원
------------------------------------
상품의 이름과 가격을 입력해주세요.
상품의 이름을 입력해주세요 : 딸기
가격을 입력해주세요 : 500
기본 재고는 10개로 설정됩니다.
```
배열에 이 입력한 값이 추가됨

---

관리자 모드에서 3번(상품 수정)을 입력하면
```
--------------상품 수정--------------
1. 사과 5000원
2. 망고 4000원
3. 토마토 2000원
------------------------------------
입력하세요 : 2
수정할 이름을 입력해주세요(현재 이름 : 망고) : 바나나
수정할 가격을 입력해주세요(현재 가격 : 4000) : 3000
망고(4000원)에서 바나나(3000원)으로 변경되었습니다.
```
배열의 값이 실제로 수정됨

---

관리자 모드에서 4번(상품 삭제)을 입력하면
```
--------------상품 삭제--------------
1. 사과
2. 망고
3. 토마토
------------------------------------
입력하세요 : 1
사과을/를 정말로 삭제하시겠습니까? (Y/N) : Y
사과가 삭제되었습니다.
```
배열의 값이 실제로 삭제됨

---
관리자 모드에서 5번(재고 추가)를 입력하면
```
--------------재고 추가--------------
1. 사과 5개
2. 망고 4개
3. 토마토 3개
------------------------------------
입력하세요 : 3
토마토는 현재 3개입니다.
얼마나 추가하시겠습니까? : 6
토마토가 6개 추가되어 9개가 되었습니다.
```
배열의 값이 실제로 수정됨

---
