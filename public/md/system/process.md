# 시스템 프로그래밍 - 프로세스

> **주의:** 실습은 Linux 환경에서 C 언어를 사용해서 진행하니, C 언어 문서와 Linux 문서를 먼저 보고 오는 것을 추천합니다.

프로세스는 **실행 중인 프로그램**이에요.
프로그램이 그냥 "파일"이라면, **프로세스는 실제로 돌아가는 상태**라고 보면 됩니다.

---
[AD]

## 1단계: 프로세스가 생기는 순간

* `chrome` 같은 프로그램을 실행하면 **프로세스가 생성**됩니다.
* 운영체제(OS)가 이 프로세스에 **CPU와 메모리**를 나눠 줍니다.

---
[AD]

## 2단계: 부모-자식 관계

프로세스는 **자식 프로세스**를 만들 수 있어요.
즉, 부모가 자식을 만드는 구조가 가능합니다.

간단한 예시 문장으로 보면:

```
부모 프로세스가 실행된다.
부모가 자식 프로세스를 만든다.
부모와 자식이 각각 실행된다.
```

---
[AD]

## 핵심 정리

* 프로세스는 **실행 중인 프로그램**
* OS가 **CPU, 메모리**를 나눠 준다
* 프로세스는 **자식을 만들 수 있다**
