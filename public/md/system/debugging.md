# 23. 디버깅 체크리스트

프로그램이 이상할 때는 아래 순서로 확인합니다.

### 23.1 컴파일 단계

```bash
gcc -Wall -Wextra -g main.c -o main
```

- 경고가 있는가?
- 헤더를 빼먹지 않았는가?
- 함수 반환값 타입이 맞는가?

### 23.2 실행 단계

```bash
./main
echo $?
```

- 종료 코드가 0인가?
- 출력 순서가 예상과 다른가?
- 부모와 자식이 모두 출력하는 코드인가?

### 23.3 프로세스 확인

```bash
ps -ef | grep main
pstree -p
```

- 프로세스가 남아 있는가?
- 좀비가 생겼는가?
- 부모-자식 관계가 예상과 같은가?

### 23.4 권한 확인

```bash
ls -l main
ls -l data.txt
```

- 실행 파일에 `x` 권한이 있는가?
- 읽으려는 파일에 `r` 권한이 있는가?
- 쓰려는 파일에 `w` 권한이 있는가?

### 23.5 시스템 호출 확인

모든 중요한 시스템 호출 뒤에는 에러 처리를 붙입니다.

```c
if (pipe(fd) == -1) {
    perror("pipe");
    return 1;
}
```
