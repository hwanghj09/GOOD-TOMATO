# 시스템 프로그래밍 - 포인터

포인터는 **주소를 저장하는 변수**예요.

* `&`는 **주소**를 가져옵니다.
* `*`는 **주소가 가리키는 값**을 읽거나 바꿉니다.

---

## 메모리 주소 예시

```
주소     값
0x0001   10   <- a가 저장된 곳
0x0002   20
0x0003   ...
```

* 변수 `a`는 **0x0001** 같은 주소에 저장됩니다.
* 포인터 `p`는 **그 주소(0x0001)** 를 저장합니다.

---

## 예시 1: 값, 주소, 포인터 한 번에 보기

```c
#include <stdio.h>
int main(){
    int a = 10;
    int *p = &a;

    printf("a value: %d\n", a);
    printf("a address: %p\n", (void*)&a);
    printf("p value: %p\n", (void*)p);
    printf("*p value: %d\n", *p);
    return 0;
}
```

### 설명

* `a value`는 **a의 값**입니다. (10)
* `a address`는 **a의 주소**입니다.
* `p value`는 **p가 가진 값**인데, 그 값은 **a의 주소**입니다.
* `*p value`는 **p가 가리키는 값**이므로 a와 같습니다.

---

## 예시 2: 포인터로 값 바꾸기

```c
#include <stdio.h>
int main()
{
    int a = 20;
    int *p = &a;

    *p = 50;

    printf("%d\n", a);
    return 0;
}
```

### 설명

* `p`는 **a의 주소**를 저장합니다.
* `*p = 50`은 **a의 값을 50으로 바꾸는 것**과 같습니다.
* 그래서 출력 결과는 **50**입니다.

---

## 핵심 정리

* 포인터는 **주소를 저장하는 변수**
* `&`는 주소, `*`는 값
* 포인터로 **값을 읽고/바꿀 수 있다**
