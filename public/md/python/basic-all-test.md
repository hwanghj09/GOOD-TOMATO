## 30. 다음 삽입 정렬(Insertion Sort) 코드의 실행 결과로 올바른 것을 고르시오.
```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

print(insertion_sort([5, 3, 1, 4, 2]))
```
1) `[5, 3, 1, 4, 2]`
2) `[1, 2, 3, 4, 5]`
3) `[5, 4, 3, 2, 1]`
4) `[2, 3, 1, 4, 5]`
5) `[1, 3, 2, 4, 5]`
