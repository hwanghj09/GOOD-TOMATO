# Windows와 차이점
[AD]

리눅스와 Windows는 사용 방식과 철학이 조금 다릅니다.

## 핵심 차이

- **파일 시스템**: 리눅스는 루트(`/`) 아래에 모든 폴더가 배치됩니다.
- **경로 표기**: 리눅스는 `/`를 사용, Windows는 `C:\`처럼 드라이브 개념이 있습니다.
- **대소문자 구분**: 리눅스는 대소문자를 구분합니다. (`File` ≠ `file`)
- **설치 방식**: 리눅스는 패키지 관리자로 설치/업데이트를 합니다.
- **권한 모델**: 사용자/그룹/기타 사용자로 권한을 관리합니다.

## 예시

- Windows: `C:\Users\name\Documents`
- Linux: `/home/name/Documents`
