# 상대 경로 vs 절대 경로

리눅스에서 경로는 **현재 위치를 기준으로 쓰는지**에 따라 나뉩니다.

## 절대 경로

- 루트(`/`)부터 시작하는 전체 경로
- 어디에서 실행하든 항상 같은 위치를 가리킴

예시
--
```
/home/user/docs/report.txt
```

## 상대 경로

- 현재 작업 디렉토리를 기준으로 한 경로
- 현재 위치가 바뀌면 의미도 바뀜

예시
--
```
./docs/report.txt
../images/logo.png
```

## 요약

- **절대 경로**: 항상 같은 위치
- **상대 경로**: 현재 위치에 따라 달라짐
